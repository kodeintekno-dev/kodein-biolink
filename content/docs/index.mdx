Selamat datang di **Kodein Project**. Di proyek ini, kamu akan belajar membuat sebuah website _Link Bio_ seperti yang biasa kamu lihat di profil Instagram atau TikTok.

Website ini akan kita buat **dari nol**, hanya dengan menggunakan **HTML dan CSS**. Kamu tidak perlu pengalaman ngoding sebelumnya — cukup ikuti langkah-langkah yang disediakan di dokumentasi ini.

## 👨‍💻 Apa yang Akan Kamu Pelajari?

- Struktur dasar HTML
- Styling halaman dengan CSS
- Membuat layout responsif untuk tampilan HP dan tablet
- Menambahkan tombol, gambar, dan link
- Mendesain halaman yang rapi dan menarik
