## Menambahkan Tombol Sosial Media

Pada langkah ini, kita akan menambahkan tombol link sosial media ke halaman Link Bio kamu.

### ðŸ”§ Tambahkan Kode HTML

1. Buka file `index.html`.
2. Salin kode berikut dan tempelkan di dalam `<body>` setelah bagian sebelumnya (misalnya setelah nama dan deskripsi):

```html
<div class="sosmed-links">
  <!-- Masukan Tombol Link Sosmed Di sini -->
</div>
```

3.  Setelah itu, tambahkan link sosial media satu per satu di dalam tag `<div class="sosmed-links">` yang sudah disalin sebelumnya.
    Berikut adalah contoh kode untuk setiap tombol link sosial media.

### ðŸŽ¨ Tambahkan Kode CSS

Sekarang buka kembali file `style.css` dan tambahkan kode berikut untuk mendesain tampilan tombol-tombol tersebut:

```css
.sosmed {
  display: flex;
  align-items: center;
  justify-content: space-between;
  background-color: white;
  color: #232227;
  padding: 10px 15px;
  border-radius: 50px;
  text-decoration: none;
  font-size: 18px;
  font-weight: bold;
  transition: 0.3s;
  box-shadow: 0 4px 10px rgba(0, 0, 0, 0.15);
}

.sosmed-icon {
  width: 50px;
}

.arrow {
  display: flex;
  justify-content: center;
  align-items: center;
  background-color: #eff2f6;
  padding: 10px;
  width: 32px;
  height: 32px;
  border-radius: 50%;
}

.arrow img {
  width: 16px;
  height: 16px;
}

.sosmed:hover {
  background-color: #01df94;
  color: white;
  box-shadow: 0 6px 15px rgba(0, 0, 0, 0.2);
}

.sosmed-links {
  display: flex;
  flex-direction: column;
  gap: 10px;
}
```

---

Dengan kode CSS ini, tombol link sosial media akan memiliki desain yang rapi dan responsif. Setiap tombol akan menampilkan ikon yang sesuai dan animasi saat dihover.
