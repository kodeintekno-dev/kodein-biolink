### 2. Buat Repository Baru

Setelah memiliki akun GitHub, langkah selanjutnya adalah membuat repository baru agar kamu bisa menyimpan kode project Link Bio-mu sendiri.

- Pertama, klik tombol **"New"** di halaman repository GitHub untuk memulai membuat repository baru.

<img
  src="/repository.webp"
  alt="Langkah membuat repository GitHub"
  style={{ width: "100%", borderRadius: "12px", marginTop: "16px" }}
/>

- Setelah itu, klik **"Import a repository"** di halaman repository GitHub untuk memulai mengimpor template project yang sudah disediakan.

  <img
    src="/import.webp"
    alt="Langkah import repository"
    style={{ width: "100%", borderRadius: "12px", marginTop: "16px" }}
  />

- **Isi kolom berikut di halaman import:**

  - **The URL for your source repository**: Masukkan URL dari repository template yang sudah disediakan:
    ```
    https://github.com/kodeintekno/biolink-project.git
    ```
  - **Repository name**: Beri nama repositori barumu, misalnya `biolink`
  - Lalu klik tombol **"Begin import"**

  <img
    src="/source.webp"
    alt="Langkah import repository"
    style={{ width: "100%", borderRadius: "12px", marginTop: "16px" }}
  />

- Setelah berhasil mengimpor repository, sekarang kamu bisa mulai mengedit dan menjalankan project-mu langsung dari GitHub menggunakan **Codespace**.

  - Buka repository yang baru saja kamu buat (contoh: `biolink`)
  - Klik tombol **"Code"**
  - Lalu pilih **"Create codespace on main"** untuk membuka editor online dari GitHub

  <img
    src="/codespace.webp"
    alt="Langkah import repository"
    style={{ width: "100%", borderRadius: "12px", marginTop: "16px" }}
  />

- Setelah berhasil mengimpor repository dan membuka project-mu di Codespace, berikut adalah struktur folder dan file yang akan kamu lihat:

  <img
    src="/codespace2.webp"
    alt="Langkah import repository"
    style={{ width: "100%", borderRadius: "12px", marginTop: "16px" }}
  />

- Agar kamu bisa melihat tampilan websitemu di browser, kita akan menggunakan ekstensi **Live Server**.

  - Di dalam Codespace, klik ikon **Extensions** di sidebar kiri
  - Ketik **"Live Server"** di kolom pencarian
  - Pilih ekstensi **Live Server** yang paling atas, lalu klik **Install**

  <img
    src="/liveserver.webp"
    alt="Langkah import repository"
    style={{ width: "100%", borderRadius: "12px", marginTop: "16px" }}
  />
